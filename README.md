# WhatsApp Automated Notification System

## Overview

This project is built using Python and Twilio API to send automated WhatsApp messages for notifications and reminders.

## Tech Stack

* Python
* Twilio API
* python-dotenv

## Features

* Send WhatsApp messages automatically
* Secure credentials using environment variables
* Easy to customize message content

## How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Create a `.env` file and add:
   ACCOUNT_SID=your_sid
   AUTH_TOKEN=your_token
4. Run the project:
   python main.py

## Use Cases

* Reminder systems
* Notification alerts
* Automated messaging
